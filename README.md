mynato
======

### client CLI tool

```sh
# add app release candidate

mynato apps:create
mynato apps:list
mynato apps:info
mynato apps:destroy
# push docker image
mynato apps:push
mynato apps:release

# app config
mynato config:list
mynato config:get
mynato config:set
mynato config:unset

mynato ps:list

mynato tags:set
mynato tags:unset
mynato tags:list

mynato releases:list
mynato releases:release

```
